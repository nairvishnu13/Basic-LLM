## How Does a Language Model Work?

This project is a short educational animation explaining the core idea
behind character-level language models.

The video visualizes:
- Vocabulary creation
- Character indexing
- Probability distributions
- Autoregressive generation

Includes:
 - LLM model implementation
 - Animation / Manim code
 - Excel calculations for reference
